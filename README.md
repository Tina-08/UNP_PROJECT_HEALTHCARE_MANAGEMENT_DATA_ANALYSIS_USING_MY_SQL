# UNP_PROJECT_HEALTHCARE_MANAGEMENT_DATA_ANALYSIS_USING_MY_SQL

I did this project in collaboration with UNP, where I designed and developed a Hospital Management System database called “Careplus” using SQL.

The main objective of this project was to build an organized and efficient system to manage various hospital operations such as maintaining patient records, doctor information, appointments, treatments, billing, and staff details. To achieve this, I created multiple interconnected tables like Patients, Medical Staff, Appointments, Treatment, Billing, Medicine, Department, and Staff, and linked them using primary and foreign keys to ensure proper data relationships and integrity.

I also inserted realistic sample data to represent actual hospital scenarios. In addition, I performed different SQL operations such as retrieving data using SELECT queries, applying conditions using WHERE clauses, and using aggregate functions like SUM, AVG, MAX, and COUNT to analyze the data. I used GROUP BY and HAVING clauses for data classification and filtering, and applied different types of joins like INNER JOIN, LEFT JOIN, and RIGHT JOIN to combine data from multiple tables for better insights.

Furthermore, I implemented queries to solve practical problems such as finding patients with multiple visits, identifying pending payments, and analyzing appointment trends. Overall, this project enhanced my understanding of database design, data management, and the practical application of SQL in handling real-world healthcare systems efficiently.
